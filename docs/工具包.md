# 專案開發工具包 (Tooling Guide)

此文件紀錄開發 `insurance_helper` 專案所需要安裝的所有工具、SDK 與相依套件。

---

## 🛠️ 開發環境與 SDK (Development Environment)

開發此專案建議安裝並配置以下核心開發工具與環境（本機各裝置可依需求安裝於自訂路徑，以下以通用與蘿蔔的桌機路徑為例）：

1. **Flutter SDK (建議版本 v3.44.6)**
   - 搭載 Dart SDK `3.12.2`。用於建立、測試及編譯 Flutter 行動應用程式。
   - 通用路徑範例：`C:\Users\<您的使用者名稱>\flutter`
   - *（蘿蔔的桌機安裝路徑為：`C:\Users\brain\flutter`）*
   - [官方網站](https://docs.flutter.dev/get-started/install)

2. **Git (建議版本 v2.20+)**
   - 用於版本控制。
   - [官方網站](https://git-scm.com/)

3. **Supabase CLI (建議版本 v2.109.1)**
   - 用於本地端測試、管理資料庫 Schema 遷移 (Migrations) 與 RLS 安全策略。
   - 通用路徑範例：`C:\Users\<您的使用者名稱>\flutter\bin\supabase.exe` (或加入 PATH 的自訂資料夾)
   - *（蘿蔔的桌機安裝路徑為：`C:\Users\brain\flutter\bin\supabase.exe`）*
   - [官方網站](https://supabase.com/docs/guides/cli)

4. **IDE (整合開發環境)**
   - 推薦使用 **Visual Studio Code** (安裝 Flutter & Dart 擴充套件) 或 **Android Studio**。

---

## 📦 專案相依套件 (Project Dependencies)

這些套件已記錄在 `pubspec.yaml` 中，若要在新環境開發，請執行 `flutter pub get` 自動安裝：

### 核心套件 (Core Packages)
* **`supabase_flutter`** (`^2.15.4`): 負責與 Supabase 後端進行連線、進行身份驗證及資料庫查詢。
* **`flutter_dotenv`** (`^6.0.1`): 用於安全載入本地環境變數 (`.env` 檔案)。
* **`shared_preferences`** (`^2.2.2`): 提供本地 Key-Value 偏好設定持久化儲存（主題顏色、深淺模式、客戶卡片/清單預設檢視）。
* **`cupertino_icons`** (`^1.0.8`): 提供 iOS 風格的圖標資產。
* **`image_picker`** (`^1.1.2`): 用於從本機相簿選取照片或拍照，支援跨平台（Web、iOS、Android 等）圖片選擇。
