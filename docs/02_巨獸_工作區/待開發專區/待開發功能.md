# 待開發功能

以下為巨獸工作區整理的未來待開發功能規劃（備用功能庫）：

1. 🕸️ **人脈拓撲網絡圖 (Relationship Topology Visualizer)**
   - **說明**：在「人脈拓撲」選單中，以互動式圖形（Node-Link Diagram）呈現客戶之間的介紹人關係與親屬關係網（例如：誰介紹了誰、家族防護網關聯）。
   - **價值**：幫助業務員快速掌握推介鏈，挖掘潛在客戶家族缺口。

2. 🎙️ **Gemini 語意語音備註提取器 (AI Voice Note Extraction)**
   - **說明**：整合手機錄音功能，將錄音檔上傳至 Supabase Storage，並串接 Gemini API 進行語音轉文字（STT）與語意分析，自動提取出時間、保單偏好、預算等重點並生成行事曆提示。
   - **價值**：拜訪完客戶後，業務員只需用語音口述，系統即會自動填寫大部分的 CRM 欄位。
