## 📝 救援日誌與事件記錄 (Incident Logs)

*當發生危機並排除後，請將過程簡短記錄於此，作為未來的自救經驗庫。*

### 範例格式：
> **2026-07-15: [範例] 更換新開發裝置環境還原失敗**
> - **問題描述**：更換新電腦後，執行 `flutter pub get` 出現相依套件版本衝突。
> - **自救處置**：透過呼叫夥伴協助確認為 Puro Flutter SDK 版本設定錯誤，已使用 `puro use stable` 對齊專案 SDK 版本，並重新安裝相依套件。
> - **狀態更新**：🟢 已排除，環境恢復正常。
