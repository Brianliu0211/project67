  - 於 `_buildWeeklyCalendarStrip` 中的週曆 Row 加入 `IconButton`（左右 chevron 圖示），點選時增減 7 天並重置畫面狀態。
  ```dart
  // 左右切換週曆的核心實作
  IconButton(
    icon: const Icon(Icons.chevron_left, color: Color(0xFF00ADB5)),
    tooltip: '上一週',
    onPressed: () {
      setState(() {
        _selectedDate = _selectedDate.subtract(const Duration(days: 7));
      });
    },
  )
  ```
