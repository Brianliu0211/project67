## 💻 技術變更明細 (Actual Technical Changes)
- **[NEW] `lib/services/app_settings.dart`**：
  - 建立 `AppSettings` 單例模式 (`ChangeNotifier`)，管理 `themeMode`、`primaryColor`、`defaultCustomerViewMode` 等偏好狀態，實作 `SharedPreferences` 本地與 Supabase User Metadata 雲端雙軌同步機制。

  ```dart
  class AppSettings extends ChangeNotifier {
    static final AppSettings instance = AppSettings._internal();
    ThemeMode get themeMode => _themeMode;
    Color get primaryColor => _primaryColor;
  }
  ```
