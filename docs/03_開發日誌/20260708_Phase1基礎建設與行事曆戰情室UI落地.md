## 💻 技術變更明細 (Actual Technical Changes)
- **[NEW] `supabase_schema.sql`**：
  - 設計並部署 `profiles`（業務員）、`customers`（客戶檔案）及 `reminders`（排程提醒）三張資料表。
  - 配置 `ON DELETE CASCADE` 級聯刪除，並配置 Row Level Security (RLS) 安全政策，限制僅能讀寫所屬業務員 `auth.uid()` 之資料。
  - 實作「Auth 註冊自動同步觸發器」，在 `auth.users` 新增帳號時自動在 `public.profiles` 同步建檔。
- **[MODIFY] `lib/main.dart`**：
  - 實作 `.env` 變數安全隔離機制，在載入失敗或檢測到 `your-project-id` 等預設占位金鑰時平滑降級至 `isOfflineMode = true` 離線預覽模式。
  - 核心偵測與平滑降級邏輯：
    ```dart
    if (supabaseUrl == null || 
        supabaseKey == null || 
        supabaseUrl.isEmpty ||
        supabaseKey.isEmpty ||
        supabaseUrl.contains('your-project-id')) {
      isOfflineMode = true;
      offlineReason = '檢測到預設金鑰占位字串';
    }
    ```
  - 配置深色質感商務主題色調，包含深藏青 (`#0D1117`)、藍綠 (`#00ADB5`)、冰藍 (`#00F5FF`) 等。
- **[MODIFY] `lib/screens/login_screen.dart`**：
  - 實作深色質感登入介面，具備 Email/密碼欄位與註冊切換。
  - 實作頂部亮橘色離線預覽橫幅 (Banner) 與「直接跳過登入」按鈕，使系統在無雲端資料庫連線的狀態下也能 100% 進行完整 UI 測試。
- **[MODIFY] `lib/screens/home_screen.dart`**：
  - 實作左側可收縮側邊欄 (Sidebar) 與 AppBar Drawer，透過媒體查詢 (`MediaQuery`) 的寬度分流 (768px)，實作 RWD 響應式佈局。
  - 實作正上方週間滾動橫式週曆，點擊可調用 `showDatePicker` 彈出式日曆選擇器。
  - 實作 06:00 - 22:00 垂直滾動時間軸，並使用 Mock Data 渲染了「穿黑色衣服」(09:00 - 10:00，打卡點樣式) 與「服學 正式活動」(14:30 - 17:30，滿版發光卡片) 二個行程卡片。
