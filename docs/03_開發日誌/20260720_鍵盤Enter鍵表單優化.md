# 📅 2026-07-20 - 【模組一點五：客戶模組與 UI 體驗優化】鍵盤 Enter 鍵提交表單優化

> **執行狀態**：🟢 已併入主線 (src)

## 🎯 實作成果摘要
- 優化客戶新增/編輯對話框與系統登入/註冊表單，使所有單行文字輸入欄位（如姓名、Email、密碼等）支援在按下鍵盤「Enter」鍵時直接提交表單，大幅提升業務員在桌機及 Web 版的操作流暢度。

## 💻 技術變更明細 (Actual Technical Changes)
- **[MODIFY] `lib/screens/customer_management_tab.dart`**：
  - 於 `_showCustomerForm` 對話框方法中，將原本「儲存」按鈕的新增與修改邏輯提取為局部函數 `void submitForm()`。
  - 為姓名、綽號、照片網址、電話、Email 及標籤等 6 個單行 `TextField` 配置 `textInputAction: TextInputAction.next` / `TextInputAction.done` 以及 `onSubmitted: (_) => submitForm()`，以支援按下 Enter 鍵提交。
  - 多行備註 `notesController` 則排除 `onSubmitted`，保留正常的換行功能。
  - 核心程式碼變更：
    ```dart
    void submitForm() {
      final name = nameController.text.trim();
      if (name.isEmpty) {
        CustomToast.show(context, '客戶姓名為必填項目', ToastType.warning);
        return;
      }
      // 依 isEdit 呼叫 _updateCustomer 或 _createCustomer...
      Navigator.pop(context);
    }
    ```

- **[MODIFY] `lib/screens/login_screen.dart`**：
  - 於登入與註冊的 `TextFormField` 設定合適的 `textInputAction: TextInputAction.next`，便於使用者按 Enter 移動焦點。
  - 為密碼欄位加上 `textInputAction: TextInputAction.done` 與 `onFieldSubmitted: (_) { if (!_isLoading) _handleSubmit(); }`，支援在密碼輸入完成後按 Enter 直接提交登入。
