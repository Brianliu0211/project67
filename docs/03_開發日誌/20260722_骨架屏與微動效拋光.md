# 📅 2026-07-22 - 【模組一與二：系統視覺拋光與排版優化】Shimmer骨架屏、Hover微特效與網頁重整狀態保存

> **執行狀態**：🟢 已併入主線 (src)

## 🎯 實作成果摘要
- 實作了 Reactbits 風格的「Shimmer 流光骨架屏」（支援 500ms 載入延遲展示）與「客戶卡片 Hover 懸停微特效」；修復了 GridView 裁剪邊緣導致卡片切斷的 Layout Bug，並導入 `SharedPreferences` 持久化當前分頁狀態，解決網頁重新整理時會被重置回首頁的痛點。

## 💻 技術變更明細 (Actual Technical Changes)
- **[NEW] [animations.dart](file:///c:/GitHub/project67/lib/widgets/animations.dart)**：
  - 建立共用動畫元件庫，實作 `ShimmerLoader`（透過 Alignment 坐標變換進行物理流光掃描）、`HoverAnimatedCard`（懸停時上浮 4px、放大 1.025x 與中心點對齊對稱放大）與 `StaggeredFadeIn`（根據 list index 依序淡入滑動）。
- **[MODIFY] [customer_management_tab.dart](file:///c:/GitHub/project67/lib/screens/customer_management_tab.dart)**：
  - 導入 `CustomerCardShimmer` 骨架與 `_buildShimmerGrid` 清單。
  - 將載入進度圈替換為骨架屏網格，並在加載函式 `_fetchCustomers` 中設定 500ms 模擬網路延遲。
  - 修正 GridView 裁剪邊界：移除 `clipBehavior: Clip.none` 以防滾動溢出蓋住上方搜尋欄，並改以 top 12px 內邊距提供 Hover 放大空間防止卡片切斷。
- **[MODIFY] [profile_screen.dart](file:///c:/GitHub/project67/lib/screens/profile_screen.dart)**：
  - 新增並整合 `ProfileShimmer` 自適應（單雙欄）骨架屏，在載入期遮蔽輸入框字體跳動現象，並設定 500ms 延遲。
- **[MODIFY] [home_screen.dart](file:///c:/GitHub/project67/lib/screens/home_screen.dart)**：
  - 於 `initState` 讀取並在分頁切換時寫入 `SharedPreferences` 中的 `'last_active_menu'`，重整網頁時自動復原選單狀態。
