## ⚠️ 環境異動與破壞性變更 (Environment & Breaking Changes)
- **套件異動**：無。
- **金鑰異動**：GitHub Actions 需要配置三筆 Repository Secrets：`VERCEL_TOKEN`、`VERCEL_ORG_ID`、`VERCEL_PROJECT_ID`。
