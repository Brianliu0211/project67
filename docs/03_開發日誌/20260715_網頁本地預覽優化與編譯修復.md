# 📅 [2026-07-15] - 【模組一：基礎建設與安全防護】本地 Web 測試版預覽優化與編譯修復

> **執行狀態**：🟢 已併入主線 (src)

## 🎯 實作成果摘要
修復了本地 Windows + Puro 開發環境下的 Flutter Web 編譯與路徑解析錯誤，建立了本地一鍵預覽腳本，並為 AI 助理新增「讓我看看」自動背景啟動功能，大幅提升團隊本地測試體驗。

## 💻 技術變更明細 (Actual Technical Changes)

- **[NEW] `run_local_web.bat`**：
  - 建立了一鍵啟動本地 Web-Server (Port 8080) 的批次腳本，動態載入使用者環境變數以繞過 Puro 快取相對路徑解析衝突：
    ```bat
    set FLUTTER_ROOT=%USERPROFILE%\.puro\envs\stable\flutter
    "%USERPROFILE%\.puro\envs\stable\flutter\bin\cache\dart-sdk\bin\dart.exe" --packages="%USERPROFILE%\.puro\envs\stable\flutter\packages\flutter_tools\.dart_tool\package_config.json" "%USERPROFILE%\.puro\shared\flutter_tools\ee80f08bbf97172ec030b8751ceab557177a34a6\flutter_tools.snapshot" run -d web-server --web-port=8080
    ```

- **[MODIFY] [main.dart](file:///c:/GitHub/project67/lib/main.dart)**：
  - 修復了 Flutter 3.44.6 編譯時 `CardTheme` 的類型不符問題（變更為 `CardThemeData`）：
    ```dart
    // 修改前
    cardTheme: const CardTheme(elevation: 2, ...),
    // 修改後
    cardTheme: const CardThemeData(elevation: 2, ...),
    ```

- **[MODIFY] [.agents/AGENTS.md](file:///c:/GitHub/project67/.agents/AGENTS.md)**：
  - 新增「讓我看看」自動化指令規則，使 AI 助理能在接收指令時自動啟動 Web Server，並告知點擊連結。

- **[MODIFY] [開發人員手冊.md](file:///c:/GitHub/project67/docs/00_公共規格/開發人員手冊.md)**：
  - 於手冊中新增「啟動本地網頁測試版預覽 (「讓我看看」)」小節，記錄 AI 自動啟動與手動批次檔一鍵啟動的使用說明。

## 🚦 團隊協作與驗證指南 (Verification Plan)
- **工程師驗飾**：直接於專案目錄中雙擊 `run_local_web.bat`，或是對 AI 輸入「讓我看看」，隨後在瀏覽器開啟 **[http://localhost:8080](http://localhost:8080)** 進行網頁測試。
